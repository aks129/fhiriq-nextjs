## Contributing

We welcome contributions from the FHIR community! Whether you're fixing typos, adding examples, or sharing your implementation experiences, your input is valuable.

### How to Contribute

1. **Fork the Repository**
   ```bash
   git clone https://github.com/aks129/fhiriq-nextjs.git
   cd fhiriq-nextjs
   ```

2. **Create a Branch**
   ```bash
   git checkout -b add-my-contribution
   ```

3. **Make Your Changes**
   - Edit guide files in `docs/guides/`
   - Add code examples, diagrams, or explanations
   - Share real-world implementation experiences

4. **Submit a Pull Request**
   - Describe what you're adding/changing
   - Reference any related issues
   - Include examples if applicable

### Contribution Guidelines

- **Be Practical**: Focus on real-world implementation experiences
- **Include Examples**: Code snippets, JSON examples, or diagrams help immensely
- **Cite Sources**: Reference official FHIR specs, IGs, or other resources
- **Keep It Current**: Use the latest FHIR versions (R4, R5)
- **Share Learnings**: Document what worked, what didn't, and why
